## 演出难度分析

- **普罗米修斯的火种** —— 难度中到高。需表现抽象象征与内心挣扎，鹰啄食场面需特殊道具。
- **珀尔修斯和美杜莎** —— 难度中。叙事清晰、动作明确；难点在美杜莎妆造与"避免直视"的走位设计。
- **伊卡洛斯的飞行** —— 难度中等。情节简单，但飞行效果需创意舞台 / 光影。
- **欧瑞狄刻和俄耳甫斯** —— 难度中到高。情感驱动，对表演与冥界氛围（灯光、音效）要求高。

!!! tip "选择建议"
    对业余选修课程，推荐情节直白、难度适中的剧目，如《珀尔修斯和美杜莎》。
